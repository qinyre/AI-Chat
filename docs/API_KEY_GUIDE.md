## 1. Google Gemini

### 特点
- ✅ **免费额度高** - 每分钟 15 次请求，每天 1500 次请求
- ✅ **无需绑卡** - 直接 Google 账号登录即可
- ✅ **申请简单** - 1 分钟完成

### 申请步骤

#### 第一步：访问 Google AI Studio

打开浏览器，访问：
```
https://makersuite.google.com/app/apikey
```

#### 第二步：登录 Google 账号

- 如果未登录，使用您的 Google 账号登录
- 没有账号需要先注册 [Google 账号](https://accounts.google.com/signup)

#### 第三步：创建 API 密钥

1. 点击 **"Create API key"** 按钮
2. 在弹出的对话框中，选择或创建一个项目
3. 复制生成的 API Key（格式：`AIzaSy...`）

#### 第四步：验证 API 密钥

API 密钥格式：`AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

## 2. DeepSeek

### 特点
- 💰 **超低价格** - 输入 ¥0.001/千 tokens，输出 ¥0.002/千 tokens
- 🚀 **性能强劲** - DeepSeek-chat 模型在开源榜名列前茅
- 💳 **需要充值** - 新用户赠送一定额度，用完需充值

### 申请步骤

#### 第一步：注册 DeepSeek 账号

1. 访问 DeepSeek 开放平台：
   ```
   https://platform.deepseek.com
   ```

2. 点击右上角 **"注册"** 按钮
3. 使用手机号或邮箱注册
4. 完成邮箱/手机验证

#### 第二步：登录并进入 API Keys 页面

1. 登录后，点击左侧菜单 **"API Keys"**
   或直接访问：
   ```
   https://platform.deepseek.com/api_keys
   ```

#### 第三步：创建 API Key

1. 点击 **"创建 API Key"** 按钮
2. 输入名称（可选），如：`AI-NEXUS`
3. 点击 **"提交"**
4. **重要**：立即复制 API Key（格式：`sk-...`）
   - ⚠️ 离开页面后将无法再次查看完整密钥

#### 第四步：充值（可选）

1. 点击左侧菜单 **"充值"**
2. 选择充值金额（建议先充值 ¥10 测试）
3. 支持微信支付、支付宝等

#### 第五步：验证 API 密钥

API 密钥格式：`sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

## 3. Moonshot / Kimi

- 🇨🇳 **中文优化** - Kimi 对中文理解能力强
- 💰 **需要充值** - 新用户赠送 ¥15 额度

### 申请步骤

#### 第一步：注册 Moonshot 账号

1. 访问 Moonshot 开放平台：
   ```
   https://platform.moonshot.cn
   ```

2. 点击右上角 **"注册"** 按钮
3. 使用手机号注册
4. 完成短信验证码验证

#### 第二步：登录并进入 API Keys 页面

1. 登录后，点击右上角头像 → **"API Keys"**

1. 点击 **"新建 API Key"** 按钮
2. 输入名称（可选），如：`AI-NEXUS`
3. 点击 **"确定"**
4. **重要**：立即复制 API Key（格式：`sk-...`）
   - ⚠️ 仅显示一次，请妥善保存

#### 第四步：充值（可选）

1. 点击左侧菜单 **"计费管理"**
2. 点击 **"充值"**
3. 选择充值金额
4. 支持微信支付、支付宝

#### 第五步：验证 API 密钥

API 密钥格式：`sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

2. 输入名称（可选），如：`AI-NEXUS`

## 5. Spark (讯飞星火)

### 特点
- 🇨🇳 **中文优化** - 讯飞星火在中文领域表现优秀
- 🎓 **多版本** - V1.5、V2、V3.0、V3.5 等版本
- ⚠️ **认证繁琐** - 需要企业认证或个人实名认证
- 🔧 **特殊格式** - API Key 格式为 `Bearer xxxxx`

### 申请步骤

#### 第一步：注册讯飞开放平台账号

1. 访问讯飞开放平台：
   ```
   https://www.xfyun.cn
   ```

2. 点击右上角 **"注册"** 按钮
3. 使用手机号注册
4. 完成手机验证

#### 第二步：实名认证

1. 登录后，点击右上角头像 → **"账号管理"**
2. 完成 **实名认证**（需要身份证）
3. 等待审核通过（通常几分钟）

#### 第三步：创建应用

1. 进入 **控制台**：
   ```
   https://console.xfyun.cn/services/cbm
   ```

2. 找到 **"讯飞星火认知大模型"**
3. 点击 **"立即开通"**
4. 点击 **"创建应用"**
5. 填写应用信息：
   - 应用名称：`AI-NEXUS`
   - 应用描述：`多模型 AI 聚合聊天平台`

#### 第四步：获取 API Key

1. 在应用列表中，找到刚创建的应用
2. 点击 **"管理"** → **"API密钥"**
3. 复制以下信息：
   - **API Secret**（格式：`xxxxxxxxxxxx`）
   - **API Key**（格式：`xxxxxxxxxxxx`）

⚠️ **注意**：Spark API 的认证方式较特殊，需要将 API Key 和 API Secret 组合使用。

#### 第五步：验证 API 密钥

在应用配置中，Spark API Key 直接使用 **API Key** 即可：
```
格式：xxxxxxxxxxxx（不是 Bearer 开头）
```

## 常见问题

### Q1: API Key 泄露了怎么办？

**A**: 立即采取以下措施：
1. 在提供商平台删除泄露的 API Key
2. 创建新的 API Key
3. 更新应用配置
4. 检查账单是否有异常消费

### Q2: 如何查看 API 使用量和费用？

**A**:
- **DeepSeek**: 控制台 → **使用记录**
- **Moonshot**: 控制台 → **用量统计**
- **Qwen (SiliconFlow)**: 账户中心 → **消费记录**
- **Google Gemini**: [Google AI Studio](https://makersuite.google.com/app/apikey)
- **Spark**: 控制台 → **账单管理**

### Q3: 免费额度用完怎么办？

**A**:
1. **充值续费** - 在各平台控制台充值
2. **更换模型** - 尝试其他有免费额度的提供商
3. **优化使用** - 减少无效请求，精简提示词

### Q4: API Key 无效/报错怎么办？

**A**: 检查以下几点：
1. ✅ API Key 是否完整复制（没有多余空格）
2. ✅ API Key 是否已激活（有些需要充值）
3. ✅ 账户状态是否正常（未被封禁）
4. ✅ 网络连接是否正常

### Q5: 哪个模型最推荐新手使用？

**A**: 推荐顺序：
1. 🥇 **Google Gemini** - 完全免费，无需绑卡
2. 🥈 **DeepSeek** - 极低价格，性能强劲
3. 🥉 **Qwen (SiliconFlow)** - 有免费额度

## 安全建议

### 🔒 保护 API Key 安全

1. **不要分享** - API Key 相当于密码，不要分享给他人
2. **不要提交** - 已添加到 `.gitignore`，不会被提交到 Git
3. **定期更换** - 建议每 3-6 个月更换一次
4. **监控账单** - 定期检查 API 使用量和费用

### 💡 费用控制建议

1. **设置预算** - 在各平台设置每日/每月消费上限
2. **使用日志** - 定期查看 API 调用日志
3. **测试优先** - 使用免费额度充分测试后再充值
